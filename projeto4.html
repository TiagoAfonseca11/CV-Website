Project Overview

This Flutter application simplifies the process of managing lost and found items within a university. Integrated with Firebase, it allows students to report, claim, and view items through a clean and responsive mobile interface. The app features secure authentication, data persistence, and automated item expiration rules.

Key Features:

  • Firebase Authentication with secure account creation and login
  • Cloud Firestore database for real-time item tracking
  • Custom UI with Flutter animations and responsive layout
  • Automatic auctioning of unclaimed items after a defined period
  • Account deletion and secure logout functionality
  • Image uploads and item categorization for fast discovery

Technologies Used:

Flutter Dart Firebase Authentication Cloud Firestore Firebase Storage

Challenges & Solutions:

User Data Security

Used Firebase Authentication and Firestore security rules to ensure users could only access their own data and actions.

Dynamic Item Management

Implemented real-time updates with Firestore listeners and timers to automatically handle expired items and trigger auctions.

Smooth User Experience

Designed a lightweight yet elegant interface with responsive layouts and custom transitions using Flutter’s animation framework.
