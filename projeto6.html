Project Overview

NBA Predictor is a championship prediction system for the NBA based on real data, season stats, clutch time performance, and historical analysis. It uses machine learning to calculate probabilities and provide insights about which teams are most likely to win the 2025-26 title.

Key Features:

  • Top 10 title contenders prediction with percentage probabilities
  • Integration of clutch stats, regular season data, and historical performance
  • Detailed team analysis including offensive rating, defensive rating, and plus/minus
  • Automatic CSV file generation with predictions
  • Visualization of the current NBA standings

Technologies Used:

Python 3.11 Pandas Scikit-learn NumPy Matplotlib

Challenges & Solutions:

Combining multiple data sources

Custom parsers were developed to merge current stats, historical data, and clutch performance, ensuring data integrity and synchronization.

Prediction accuracy

The model was fine-tuned using data from the past 5 seasons and validated against historical outcomes to improve accuracy and reduce false positives.
