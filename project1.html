Project Overview

This project is a high-performance image converter that transforms PNG files into scalable SVG vector graphics.
This helps designers and everyone that uses social media, websites or even emails to convert images that have no scalability into images that are vectorial, loosing none of it's quality.

Key Features:

  • Supports batch processing of multiple files
  • Preserves transparency and color accuracy
  • Customizable output resolution
  • Command-line interface for easy integration into workflows
  • Optimized memory management for large image files

Technologies Used:

C++17 Standard Template Library (STL) libpng Custom vectorization algorithms

Challenges & Solutions:

Color Accuracy Preservation

Implementing precise color mapping from raster to vector format required developing custom quantization algorithms to maintain visual fidelity while optimizing file size.

Performance Optimization

By utilizing multi-threading and efficient memory management techniques, the converter can process large batches of images with minimal resource usage.
