Project Overview

This full-stack freelance marketplace allows clients to post job offers and hire service providers seamlessly.
Developed using PHP with an SQLite database, the platform integrates a secure backend API, regex-based validation for payments, and robust data protection via prepared statements. The platform includes messaging, transaction handling, and filtering mechanisms for a smooth user experience. The project was developed with 2 classmates at university.

Key Features:

  • User registration and login with encoded characters and input sanitization
  • Client-freelancer messaging system scoped to each transaction
  • Simulated payment system with regex validation for card formats
  • Protected SQL queries using prepared statements
  • Advanced project filtering and search capabilities
  • Admin panel for category and user management

Technologies Used:

PHP SQLite JavaScript HTML5 CSS RESTful API Regex Prepared Statements

Challenges & Solutions:

Security & SQL Injection Protection

All database interactions were implemented using prepared statements to eliminate the risk of SQL injection, ensuring safe input handling throughout the app.

Transaction Communication Flow

To ensure clear communication between users, each project transaction has an isolated message thread, allowing client-provider coordination without data overlap.

Validation of Simulated Payments

Regex was used to verify card details format during checkout simulations, maintaining both usability and basic validation logic without external libraries.
