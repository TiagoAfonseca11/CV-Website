Project Overview

This project is a low-level implementation of the classic Minesweeper game, developed in C using the Minix3 operating system.
It fully integrates hardware-level control of input and output devices, showcasing direct interaction with the keyboard, mouse, real-time clock (RTC), and graphics card. Developed as part of a university assignment alongside classmates for the course "Computer Labs".

Key Features:

  • Fully interactive Minesweeper gameplay
  • Mouse and keyboard input handling through hardware interrupts
  • Graphical interface rendered via direct video memory access
  • Custom timer and RTC integration for game timing and events
  • Modular and maintainable codebase in pure C

Technologies Used:

C Minix3 i8254 Timer i8042 Keyboard & Mouse VBE Graphics Mode RTC (Real-Time Clock)

Challenges & Solutions:

Hardware Interaction

Accessing and controlling low-level devices like the mouse, RTC, and keyboard required deep understanding of hardware interrupts and Minix3's system calls.

Graphics Programming

Rendering a functional and responsive UI involved direct manipulation of video memory using VBE, without relying on any high-level graphics library.
